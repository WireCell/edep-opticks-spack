#+title: Spack packages for edep-sim + eic-optics 

* Quick start

#+begin_example
git clone https://github.com/WireCell/edep-opticks-spack.git
spack repo add edep-opticks-spack/edep_opticks/

spack info edepsim
spack info eicoptics

spack install edepsim
#+end_example
